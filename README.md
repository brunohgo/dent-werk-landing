# Dent Werk | Studio de Prótese Sobre Implante

Landing page premium para o Dent Werk - especialista em próteses fixas sobre implantes em Lisboa.

**Stack:** Next.js 16 + TypeScript + Tailwind + Framer Motion + React Three Fiber (3D interativo) + i18n (PT/EN/ES)

## Como rodar localmente

```bash
npm install
npm run dev
```

## Deploy

Deploy automático na Vercel ao fazer push para `main`.

Design premium, experiência 3D imersiva e totalmente traduzida em Português de Portugal, Inglês e Espanhol.
